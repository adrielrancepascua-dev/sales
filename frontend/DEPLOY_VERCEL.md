Vercel Deploy Steps

1. Open terminal in frontend folder.
2. Run: npx vercel login
3. Run: npx vercel
4. For production: npx vercel --prod

Recommended settings for prompts
- Framework preset: Other
- Root directory: .
- Build command: (leave empty)
- Output directory: (leave empty)

After deploy
1. Open the live URL on phone.
2. Switch to Owner Mode.
3. Open Settings and fill:
   - Business Display Name
   - Business ID (BIZ-001)
   - Webhook URL
   - API Key
   - Owner Access Code
4. Save Settings.
5. Start a cashier shift and test one sale.
